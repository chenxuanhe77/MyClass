# MyClass
##### 使用罗鑫提供的接口开发一款关于本班信息的app
* 功能有：
    * 课表查询
    * 上课通知
    * 违纪查询
    * 校园卡消费记录及余额查询
    * 本班通讯录等
    * 更多功能正在开发中  
* 只面向本班用户
